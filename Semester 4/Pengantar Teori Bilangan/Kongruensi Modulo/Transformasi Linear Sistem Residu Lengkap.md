#Teorema 

Misalkan $S$ membentuk Sistem Residu Lengkap modulo $n$. Definisikan 
$$H = aS + b = \{ as + b \ : \ s \in S \}$$
$H$ membentuk Sistem Residu Lengkap modulo $n$ jika dan hanya jika $(a, n) = 1$
